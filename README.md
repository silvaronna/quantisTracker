# Quantis Track 📈

A modern, high-performance portfolio tracking application built with Flutter. Quantis Track provides real-time market insights with a focus on clean aesthetics and interactive data visualization.

## 🚀 Features

- **Interactive Market Charts**: Real-time price tracking with hover-enabled tooltips showing price, date, and time.
- **Portfolio Management**: Comprehensive overview of asset performance, average buy prices, ROI, and allocations.
- **Neon Dark UI**: A sleek, high-contrast dark theme with neon teal accents for maximum readability.
- **Asset Filtering**: Quick search and dropdown selection for top assets like Bitcoin, Ethereum, and Apple INC.
- **Responsive Design**: Optimized for mobile and web views with a consistent, modern experience.

## 🛠 Tech Stack

- **Framework**: [Flutter](https://flutter.dev/) (Dart)
- **Charts**: [fl_chart](https://pub.dev/packages/fl_chart)
- **UI Components**: Custom Neon-style widgets
- **Preview**: [Device Preview](https://pub.dev/packages/device_preview)
